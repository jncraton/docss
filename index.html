    Overview

    Common semantic block elements, such as paragraphs are supported. Inline elements such as emphasis, strong, code(), highlight, and links can be used as well.

    Semantic elements with specific use cases, such as abbr elements and dates (since Oct 3, 2025) are supported.

    Blockquotes are styled appropriately if included.

    — Someone Famous

    Code

    We can include additional paragraphs. These paragraphs may include inline code elements. In order to include full block of code, you can wrap a code element within a pre element: 

    <pre>
  <code>
    if i % 3 == 0:
      print("foo")
    else:
      print("bar")
  </code>
</pre>
